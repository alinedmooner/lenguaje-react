# Lenguaje React

Esta página web tiene como finalidad ser un proyecto que permita tener un entendimiento de todos los conceptos, tecnicas, recursos, librerías, etc en el ecosistema de la librería ReactJS.

## Estructura

La página será un blog con entradas dividas por categorías o grupos de temas similares, la idea es esquematizar correctamente los conceptos, definirlos correctamente y servir de soporte para el desarrollo de aplicaciones con React.
